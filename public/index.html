사용법

  • 게이트 추가: 버튼을 클릭한 후 캔버스를 클릭
  • INPUT 토글: INPUT 게이트를 클릭하여 0/1 전환
  • 연결: 출력 포트를 우클릭하고 입력 포트로 드래그
  • 이동: 게이트를 드래그하여 이동
  • 화면 이동: 마우스 휠 클릭 후 드래그
  • 확대/축소: + - 버튼으로 화면 크기 조절
  • 보기 모드: INPUT 값 변경만 가능 (추가/이동/삭제 불가)
  • 프리셋: 기본 회로를 캔버스에 생성
  • 삭제: 삭제 모드에서 게이트 클릭
